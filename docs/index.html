Basic Keylogger

A modern & ethical Python keylogger project for learning and security auditing.

Get Started

Features

Comprehensive Logging

Records every keystroke, including special keys, with session separation and timestamps.

Secure & Ethical

For educational or authorized use only, with clear session IDs and logs stored locally.

Cross-Platform

Works on Windows, Linux, and macOS using pynput library.

Easy Setup

Simple installation and execution. Start logging keystrokes in just a few commands.

Live Session Control

Auto-flushes logs, and ends cleanly with the ESC key. Each session gets a unique log file.

Organized Logs

Logs are saved in a structured logs/ folder, with one file per session.

Usage

1 Clone: git clone https://github.com/Tanvi228-yadav/basic-keylogger.git
2 Install: pip install -r requirements.txt
3 Run: python keylogger.py
4 Stop: Press ESC key
5 View Logs: Open logs/ and check the latest log_*.txt
