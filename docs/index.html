Welcome!

This project demonstrates a simple, ethical, and educational keylogger written in Python.
Warning: Only use with explicit permission and for learning or authorized auditing!

Features

  • Logs all keystrokes (characters and special keys)
  • Saves logs to uniquely named files for each session
  • Cross-platform: Windows, Linux, macOS
  • Easy setup and usage
  • Session management and buffer auto-flushing
  • Clean exit with ESC key

How to Use

  1. Clone the repository:
    git clone https://github.com/Tanvi228-yadav/basic-keylogger.git
  2. Install requirements:
    pip install -r requirements.txt
  3. Run the keylogger (for authorized testing only):
    python keylogger.py
  4. Stop the keylogger:
    Press ESC.
  5. View logs:
    Open the logs/ folder and view the latest log file.
